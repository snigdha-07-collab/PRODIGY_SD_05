# 🌐 Smart Product Scraper Pro

A Python-based web scraping application that extracts product details such as names, prices, and ratings from an e-commerce website and stores the data in a CSV file. The project features a simple and user-friendly GUI built using Tkinter.

## 🚀 Features
- Product data extraction
- GUI-based interface
- CSV export functionality
- Data display in table format
- Error handling

## 🛠️ Technologies Used
- Python
- Tkinter
- Requests
- BeautifulSoup4
- Pandas

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python web_scraper_gui.py
